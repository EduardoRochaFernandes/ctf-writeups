First-year Cybersecurity student documenting my progression through TryHackMe's SOC Analyst learning paths. Notes taken in Notion during each room, expanded into structured writeups here. ## ✍️ Writeup Philosophy

Writeups focus on **understanding and real-world application**, not just answers. Each one covers:
- What the room is about and why it matters for SOC work
- Key concepts in my own words
- Tools used and specific commands/filters that are actually useful
- Takeaways I'd apply in a real environment

Flags and direct challenge answers are never included.

## ⚠️ Note on Respect for the Platform

This repository documents learning and understanding. No flags, direct solutions, or content that would undermine the experience for other TryHackMe users is included.
