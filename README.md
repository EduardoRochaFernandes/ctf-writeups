# CTF Writeups and Learning Journal

Personal documentation of cybersecurity learning through TryHackMe structured paths and Blue Team Labs Online. This repository tracks progression across multiple learning paths with detailed notes, practical exercises, and key takeaways from each completed room.

The content is oriented towards three complementary roles: SOC analyst operations, blue team defence engineering, and security engineering — reflecting the breadth of skills required to work effectively in a modern security operations environment.

---

## Platforms

**TryHackMe** — Primary learning platform. Structured paths with guided rooms, labs, and challenges. This repository tracks four paths: SOC Level 1, SOC Level 2, SOC Level 3 (Advanced Endpoint Investigations), Security Engineer, and DevSecOps.

**Blue Team Labs Online** — Scenario-based investigations using real log data and artefacts. Planned for integration alongside TryHackMe coverage to reinforce analytical skills in unguided conditions.

---

## Repository Structure

```
ctf-writeups/
├── soc-level-1/          TryHackMe SOC Level 1 — 14 sections, 67 rooms
├── soc-level-2/          TryHackMe SOC Level 2 — 8 sections, ~40 rooms
├── soc-level-3/          TryHackMe SOC Level 3: Advanced Endpoint Investigations
├── security-engineer/    TryHackMe Security Engineer path
├── devsecops/            TryHackMe DevSecOps path
└── supplementary/        Rooms completed outside official paths
```

Each room directory contains a single `README.md` with structured notes: overview, key concepts, practical commands or techniques, and takeaways relevant to real-world work.
