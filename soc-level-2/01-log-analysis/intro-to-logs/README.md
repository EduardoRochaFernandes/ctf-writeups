## Key Notes

Log fundamentals: timestamps (always UTC, NTP sync critical), log sources (physical: access control; virtual: OS, network devices, applications, cloud), log formats (NCSA CLF for Apache, Syslog, Windows EVTX, JSON). Storage tiers: hot (0-6 months, near real-time), warm (6-24 months), cold (2-5 years, archival). Log analysis pipeline: Parsing, Normalisation, Enrichment, Correlation, Visualisation. Super timeline concept: aggregating logs from multiple sources into a single chronological view using tools like Plaso.
