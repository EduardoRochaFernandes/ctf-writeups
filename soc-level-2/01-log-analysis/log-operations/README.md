## Key Notes

Windows Event Log structure and critical Event IDs: 4624/4625 (logon success/failure), 4688 (process creation), 4698 (scheduled task creation), 1102 (security log cleared). Linux log locations: `/var/log/auth.log`, `/var/log/syslog`. Command-line analysis tools: grep, cut, sort, uniq -c, awk, wc. PID and TID for process correlation.
