## Key Notes

Cybersecurity career landscape: SOC Analyst (entry via Security+, BTL1, CySA+), Penetration Tester (OSCP, CEH), Security Engineer (GCIA, GCIH), Incident Responder (GCIH, GCFE), Malware Analyst (GREM), Threat Intel Analyst (GCTI). The SOC provides the broadest early-career exposure to all attack types across every technology layer.
