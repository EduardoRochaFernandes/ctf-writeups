## Key Notes

CMD vs PowerShell capabilities. PowerShell for analysts: `Get-NetTCPConnection` for active connections, `Get-WinEvent` for event log queries, `Get-ScheduledTask` for persistence checking, `Get-ItemProperty HKLM:\...\Run` for Run key inspection. PowerShell remoting with `Invoke-Command` and `Enter-PSSession` for remote investigation.
